   ```
   $ streamlit run streamlit_app.py
   ```

### Customization

To personalize this portfolio:

1. Update the project data in `streamlit_app.py`
2. Replace placeholder images with your actual project screenshots
3. Modify the timeline events to reflect your learning journey
4. Update contact information in the footer
5. Customize colors and styling in the CSS section
